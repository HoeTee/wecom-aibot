# 文档写作

## 默认结构

除非用户明确要求其他形式，生成型总结文档默认采用以下结构：

1. 背景
2. 逐篇总结
3. 对比分析
4. 结论或建议

## 必须满足

- 标题层级清晰
- 段落简洁
- 适合扫描阅读时使用 bullet points
- 涉及多份来源材料时必须显式比较

## 绝对不要

- 一大段无法阅读的长文本
- 空泛的 executive language
- 直接复制原始材料而不做综合
- 静默忽略用户要求的某一份来源材料

## 编辑规则

如果用户要求“补充”“追加”“按某个结构重写”或“修改上一个文档”，默认优先编辑已有文档；只有用户明确要求新文档时，才创建新文档。
